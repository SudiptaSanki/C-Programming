# Phase 1: The Foundations (School Level & Absolute Beginners)

Welcome to **Phase 1** of the C Programming Masterclass! 

Whether you are a school student writing your first line of code, or a college student looking to solidify your fundamentals, this phase is designed for you. Here, we assume **zero prior programming experience**. By the end of this phase, you will understand how a computer reads your code, how to store and manipulate data, and how to write programs that can make decisions and repeat tasks.

We will bridge the gap between textbook theory and actual coding, culminating in a milestone project that will sharpen your algorithmic thinking.

---

## 📚 Module Overview

### Module 1: Introduction & Environment Setup
Before we write code, we need to understand the tools of the trade.
* **The Story of C:** Why a language created in the 1970s still powers modern operating systems and embedded devices.
* **Under the Hood:** How compilation works (Source Code $\rightarrow$ Preprocessor $\rightarrow$ Compiler $\rightarrow$ Assembler $\rightarrow$ Linker).
* **Workspace Setup:** Installing the GCC compiler, configuring VS Code, and writing your very first `Hello, World!` program.

### Module 2: Variables, Data Types & I/O
Learn how to store information and interact with the user.
* **Primitive Data Types:** Understanding `int`, `float`, `char`, and `double`, along with how much memory they consume.
* **Formatting:** Mastering format specifiers (like `%d`, `%f`, `%c`) and escape sequences.
* **Input/Output:** Taking user input safely using `scanf`, printing with `printf`, and handling common input buffer issues.

### Module 3: Operators & Expressions
Time to do some heavy lifting and manipulate our data.
* **The Basics:** Arithmetic, Relational, and Logical operators.
* **Bitwise Operations:** A sneak peek into how computers process data at the bit level (AND, OR, XOR, shifts).
* **Typecasting:** How to safely convert one data type into another (implicit vs. explicit conversion).

### Module 4: Control Flow (Branching & Looping)
Programs are useless if they can't make decisions. Here, we give our code a "brain."
* **Branching:** Making choices using `if`, `else if`, `else`, the ternary operator, and `switch-case` statements.
* **Looping:** Repeating tasks efficiently using `for`, `while`, and `do-while` loops.
* **Loop Control:** Mastering the `break` and `continue` statements to manipulate loop execution.

---

## 🏆 Milestone 1: The "Number Cruncher" (Algorithms & Logic)

At the end of Module 4, it is time to put everything you've learned to the test. Programming is not just about knowing syntax; it's about solving problems. 

**The "Number Cruncher"** is a collection of classic mathematical and logical programs designed to build your algorithmic thinking. These are exactly the types of problems asked in school exams, college vivas, and basic technical interviews.

**What you will build:**
* **Basic Logic:** Even/Odd checker, Leap Year calculator.
* **Mathematical Sequences:** Factorial calculator, Fibonacci sequence generator.
* **Number Theory:** Programs to identify Prime, Twin Prime, Armstrong, Perfect, Strong, and Palindrome numbers.
* **Pattern Printing:** Designing structural patterns like Pyramids, Diamonds, and Pascal’s Triangle using nested loops.
