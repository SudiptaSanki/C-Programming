# Phase 1: The Foundations (School Level and Absolute Beginners)

Welcome to Phase 1 of the C Programming Masterclass.

This phase is for learners with zero prior programming experience. By the end of this phase, you will understand how C code turns into machine instructions, how to store and process data, and how to write programs that make decisions and repeat tasks.

---

## Who This Phase Is For

- School students starting programming from scratch.
- College beginners who want strong fundamentals.
- Self-learners who want clear, practical C basics.

## What You Will Achieve

By completing this phase, you will be able to:

- Explain the C compilation process.
- Write and run C programs using GCC.
- Use variables, data types, and formatted input/output.
- Build expressions using arithmetic, relational, logical, and bitwise operators.
- Use conditional statements and loops to control program flow.
- Solve foundational logic problems commonly asked in exams and interviews.

- Complete exercises at the end of every module.

## Exit Criteria for Phase 1

You are ready for the next phase when you can:

- Write a C program from problem statement to output.
- Trace your code manually for sample inputs.
- Debug common input/output and loop errors.
- Explain your solution clearly to another beginner.
